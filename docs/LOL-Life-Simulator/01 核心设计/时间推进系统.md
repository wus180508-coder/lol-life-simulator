# 一、系统定位

## 核心目标

时间系统用于驱动整个电竞人生的发展。

连接：

- 行动系统
    
- 事件系统
    
- 比赛系统
    
- 荣誉系统
    
- 人生阶段系统
    

形成完整循环。

---

## 核心理念

时间单位采用：

```Plain
1个月 = 1个行动回合
```

但比赛不再作为行动选择。

而是：

```Plain
行动
↓
时间推进
↓
赛程自动结算
↓
事件触发
↓
进入下个月
```

---

# 二、时间架构

## 基础时间单位

```Plain
1个月 = 1回合
```

玩家每个月：

- 选择一次核心行动
    
- 获得属性变化
    
- 自动结算当月赛程
    
- 自动触发事件
    
- 进入下个月

# 三、行动阶段


# 四、月度自动结算

- 教练表扬你
    
- 队友主动找你双排

# MVP版本原则

比赛不再作为行动。

行动系统仅保留：

```Plain
训练
Rank
复盘
社交
休息
```

比赛由赛程自动驱动。

关键赛事单独展示。

普通赛事自动汇总。

这样既保证：

```Plain
职业选手比赛很多
```

又不会让玩家：

```Plain
每周打一场比赛
点到手抽筋
```

同时也能很好地兼容梦想者、挑战者和职业者三个阶段。
