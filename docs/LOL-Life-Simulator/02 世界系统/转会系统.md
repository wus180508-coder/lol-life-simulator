# Demo版建议

先不要做复杂合同系统。

例如：

```Plain
合同年限

买断费

违约金

经纪人
```

全部删除。

---

V1只保留：

```Plain
报价
↓
选择
↓
加盟
```

即可。

这样转会系统会非常符合你们目前的人生模拟器定位：

```Plain
这不是经营游戏的交易市场

而是职业生涯中的一次人生选择
```

而且它能直接连接你们已经完成的：

```Plain
战队系统
↓
队友系统
↓
教练系统
↓
宿敌系统
↓
传奇系统
```

成为职业者阶段和传奇者阶段最重要的年度决策节点之一。
