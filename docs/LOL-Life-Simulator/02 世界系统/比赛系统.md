# 一、设计目标

## 目标1

让玩家理解：

```Plain
为什么赢
为什么输
```

---

## 目标2

让每场重要比赛都有故事

而不是：

```Plain
系统判定获胜
```

---

## 目标3

让比赛成为：

```Plain
属性验证器
```

而不是小游戏。

---
