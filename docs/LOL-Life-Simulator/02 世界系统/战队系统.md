# 系统定位

## 战队是什么

对于竞男人生模拟器：

```Plain
战队
=
玩家当前的人生环境
```

类似于：

```Plain
人生重开模拟器
→ 家庭

中国式家长
→ 学校

竞男人生模拟器
→ 战队
```

---

# 战队系统核心循环

```Plain
加入战队
↓
队友互动
↓
训练
↓
比赛
↓
成绩变化
↓
战队变化
↓
转会/留队
↓
进入新赛季
```

