# 一、系统定位

## 核心目标

生成：

```Plain
属于这一局人生的电竞时代
```

而不是：

```Plain
随机职业选手
```

---

系统负责生成：

```Plain
职业选手
↓

战队阵容

↓

联赛格局

↓

宿敌

↓

世界赛竞争者

↓

历史第一人竞争者
```

---

最终形成：

```Plain
每一局人生
=
一个新的电竞时代
```

---

# 最终生成公式

```Plain
年龄
+
位置
+
五维属性
+
潜力
+
成长曲线
+
选手原型
+
人格标签
+
时代之子标记
+
宿敌生成
=
AI职业选手
```

相比V1版本，这版最大的变化是：

```Plain
不再模拟整个电竞世界

而是模拟一个电竞时代
```

这会极大降低开发成本，同时保留《竞男人生模拟器》最重要的体验：

```Plain
队友

宿敌

冠军

王朝

历史第一人之争
```

而这四件事，已经足够支撑整个游戏的叙事闭环。
