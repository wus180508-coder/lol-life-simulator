# 一、系统定位

## 教练是什么

对于本游戏：

```Plain
队友
=
横向关系

教练
=
纵向关系
```

---

教练承担：

```Plain
培养者
评价者
决策者
人生导师
```

四重身份。

---

```Plain

↓

↓

↓
```

```Plain
完整教练NPC
```

# 最终生成公式

```Plain
年龄
+
出身
+
五项执教属性
+
执教风格
+
人格标签
+
培养倾向
=
完整教练NPC
```

这样生成出来的就不是：

```Plain
教练A
战术85
```

而是：

```Plain
31岁

前职业打野

伯乐型

偏爱努力选手

培养能力92

与你关系85

曾力保你首发
```

这种玩家会真正记住的角色。

而且这个系统还能自然连接：

```Plain
挑战者阶段
（首发竞争）

职业者阶段
（成长突破）

传奇者阶段
（王朝建立）

传承线
（未来成为教练）
```

形成完整的电竞生态闭环。
