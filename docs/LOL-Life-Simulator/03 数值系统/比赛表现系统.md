# MVP版本建议（非常重要）

我建议你们第一版不要生成：

```Plain
补刀

经济

小龙

大龙

推塔
```

这些MOBA细节。

直接使用：

```Plain
比赛评分
（0~100）
```

作为核心中间层。

即：

```Plain
比赛评分
↓
生成KDA
↓
生成MVP
↓
生成新闻
↓
生成舆论
```

因为《竞男人生模拟器》的核心是：

```Plain
人生模拟
而不是比赛模拟
```

这样比赛系统既足够真实，又不会把开发量变成一个新的LOL。
