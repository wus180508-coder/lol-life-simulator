# 五、人气衰减系统

现实里的人气不会永久存在。

因此需要衰减机制。

---

每年自动衰减：

```Plain
当前人气 × 5%
```

---

例如：

```Plain
600

↓

570
```

---

连续获得荣誉：

可以抵消衰减。

---

# 六、人气与影响力

影响力不再单独成长。

由：

```Plain
荣誉
+
人气
```

共同决定。

---

公式：

```Plain
影响力

=
荣誉值 ×60%
+
人气 ×40%
```

---

举例：

---

选手A

```Plain
荣誉 500

人气 200
```

影响力：

```Plain
380
```

---

选手B

```Plain
荣誉 500

人气 800
```

影响力：

```Plain
620
```

---

两人冠军一样。

历史评价不同。

# Demo版最终建议

V1阶段不要设计：

```Plain
粉丝数

微博热度

直播人气

商业价值

赞助收入
```

全部删除。

只保留：

```Plain
人气
（0~1000）
```

作为唯一公众影响力指标。

这样整套系统会非常简洁：

```Plain
荣誉
+
人气
=
影响力

影响力
+
荣誉
=
传奇值
```

既符合真实电竞世界的逻辑，也能很好地支撑你们后续的传奇者阶段、历史地位系统和结局系统。
