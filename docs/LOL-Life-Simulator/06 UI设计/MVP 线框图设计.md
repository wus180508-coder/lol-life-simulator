
# 一、整体设计原则

## Principle 1

人生仪表盘（Dashboard）

而不是：

```
网页应用
```

---

玩家打开游戏后：

应该看到：

```
我的人生
```

而不是：

```
一个后台管理系统
```

---

## Principle 2

信息密度高

参考：

```
Football Manager
```

---

尽量做到：

```
一屏完成主要操作
```

---

不要：

```
疯狂跳页面
```

---

## Principle 3

事件驱动

整个游戏的核心：

```
行动↓事件↓比赛↓成长
```

所以：

```
事件
```

必须成为视觉中心。
