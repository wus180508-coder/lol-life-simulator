# 事件弹窗流程

```
事件触发↓显示事件标题与描述↓玩家选择 A / B / C↓结算属性变化↓关闭弹窗↓继续月度结算
```

---

# 比赛结果弹窗流程

```
本月有比赛↓自动结算比赛↓显示：- 对阵- 胜负- 比分- 玩家评分- KDA- 奖励变化↓关闭弹窗↓回到主页
```

---

# 阶段晋升弹窗流程

```
职业关注度 ≥ 100↓梦想者 → 挑战者或首发评分 ≥ 100↓挑战者 → 职业者
```

弹窗显示：

```
人生阶段晋升新的身份新的目标
```
